Engineering thesis - a web service supporting the operation of a dental office from the perspective of four users.
The backend was created with the Java programming language and the Spring Boot framework.
The graphical user interface was designed using HTML and CSS with the Thymeleaf component.
Login is secured thanks to the Spring Security component.
The database was created in MySQL.

Patients:
- ability to browse the dental office's offer along with the price list and contact tab

Dentists:
- ability to add orders for a dental technician
- ability to add materials to the online warehouse to track inventory in the office

Dental technicians:
- ability to view orders added by the doctor and to modify the status

Administrator:
- ability to delete orders
- the ability to modify the amount or remove selected materials at the doctor's request
